
产品功能：从门店附近30米醒目标志位置，通过实景导航引导到店
制作流程是三要素： 初始醒目标志、转折/拐角标志、目标门店
目标门店：
    1、优质：TJ小馆，在写字楼上，将站在楼下的顾客根据图的指示进到门店。（商场内部非出入口门店均属于此类）
    2、普通：白日梦想家，街边拐角处，微指引
    3、疑虑：愉市，多个入口，无法预估实际效果

初始醒目标志：
    1、品牌连锁店：麦当劳、星巴克、肯德基、德克士等
    2、标志建筑
    3、小区、地铁入口


转折/拐角标志：
    实景导航后，在沿途的拐角处最醒目、特别的点
    1、颜色：颜色非常特别的招牌，如白日梦旁边的烟酒
    2、牌匾：沿街指示路牌、小区招牌、公交站牌等

建议实景数量：5步以内
